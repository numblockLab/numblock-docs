কোনো লেনদেন চেকপয়েন্ট করা হয়েছে কিনা তা জানতে `isCheckPointed`পদ্ধতি ব্যবহার করা যেতে পারে।

```
const isCheckPointed = await posClient.isCheckPointed(<tx hash>);
```

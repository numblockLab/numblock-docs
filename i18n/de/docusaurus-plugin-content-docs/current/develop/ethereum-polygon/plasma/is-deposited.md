# Wurde eingezahlt {#isdeposited}

Die `isDeposited`-Methode kann verwendet werden, um zu erfahren, ob eine Einzahlung abgeschlossen wurde.

```
const isDeposited = await plasmaClient.isDeposited(<deposit tx hash>);
```

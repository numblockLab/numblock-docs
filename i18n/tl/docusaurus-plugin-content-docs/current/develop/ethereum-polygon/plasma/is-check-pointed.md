# isCheckPointed {#ischeckpointed}

Maaaring gamitin ang `isCheckPointed`method upang malaman kung na-checkpoint ang isang transaksyon.

```
const isCheckPointed = await plasmaClient.isCheckPointed(<tx hash>);
```

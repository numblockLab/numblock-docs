# Mahahalagang kontrata {#important-contracts}

## State Syncer {#state-syncer}

Ginagamit ang kontratang ito upang ihatid ang mga pagbabago ng estado mula sa Ethereum mainchain patungo sa Bor. Nakikinig ang Heimdall sa mga `StateSynced`kaganapan sa `StateSender`kontrata at inihahatid ang mga ito sa Bor gamit ang system call.

```jsx
contract StateSender {
	function syncState(address receiver, bytes calldata data)
    external
	{
	    counter = counter.add(1);
	    emit StateSynced(counter, receiver, data);
	}
}
```

## Tagapamahala ng deposito {#deposit-manager}

Para sa pagdedeposito ng mga asset mula sa mainchain hanggang Bor. Ang kontratang ito ay gumaganap bilang isang escrow contract kung saan ang mga asset ay hawak hanggang sa ma-withdraw ang mga ito (sa pamamagitan ng withdraw manager at predicates).

```jsx
function depositERC20ForUser(address _token, address _user, uint256 _amount)
		public
{
    require(
        IERC20(_token).transferFrom(msg.sender, address(this), _amount),
        "TOKEN_TRANSFER_FAILED"
    );
    _createDepositBlock(_user, _token, _amount);
}

function _createDepositBlock(
    address _user,
    address _token,
    uint256 _amountOrToken,
    uint256 _depositId
) internal {
    ...
    stateSender.syncState(
        childChain,
        abi.encode(_user, _token, _amountOrToken, _depositId /* sequential ID */)
    );
		...
}
```

## ChildChain (Bor) {#childchain-bor}

Ang mga depositong ito mula sa itaas ay inilalagay sa mga kontrata ng token ng ERC20/721 sa bor. Nangyayari ito sa pamamagitan ng [state receiver contract](https://www.notion.so/maticnetwork/Bor-Overview-c8bdb110cd4d4090a7e1589ac1006bab#aa94e6a9373943068b93d2c0e7f3d2e6) na tumatawag sa`onStateReceive` metodong sa`ChildChain`.

```jsx
contract ChildChain {
	...
	function onStateReceive(
	  uint256, /* id */
	  bytes calldata data
	) external onlyStateSyncer {
	  (address user, address rootToken, uint256 amountOrTokenId, uint256 depositId)
			= abi.decode(data, (address, address, uint256, uint256));
	  depositTokens(rootToken, user, amountOrTokenId, depositId);
	}

	function depositTokens(
	  address rootToken,
	  address user,
	  uint256 amountOrTokenId,
	  uint256 depositId)
	{
		...
		if (isERC721[rootToken]) {
        obj = ChildERC721(childToken);
    } else {
        obj = ChildERC20(childToken);
    }
    obj.deposit(user, amountOrTokenId);	
	}
}
```

## ChildERC20/721 {#childerc20-721}

```jsx
contract ChildERC20 is ERC20 {
	function deposit(address user, uint256 amount) public onlyOwner {
	  ...
    _mint(user, amount);
		...
  }

	function withdraw(uint256 amount) public payable {
    ...
    _burn(user, amount);
		...
	}
}
```

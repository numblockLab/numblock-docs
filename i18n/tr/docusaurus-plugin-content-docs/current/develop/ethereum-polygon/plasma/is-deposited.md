# isDeposited {#isdeposited}

`isDeposited` metodu bir fon yatırma işleminin tamamlanıp tamamlanmadığını anlamak için kullanılabilir.

```
const isDeposited = await plasmaClient.isDeposited(<deposit tx hash>);
```

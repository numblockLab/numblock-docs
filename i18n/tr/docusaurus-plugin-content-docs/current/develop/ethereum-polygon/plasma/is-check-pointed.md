# isCheckPointed {#ischeckpointed}

`isCheckPointed` metodu bir işlem için denetim noktası belirlenip belirlenmediğini anlamak için kullanılabilir.

```
const isCheckPointed = await plasmaClient.isCheckPointed(<tx hash>);
```

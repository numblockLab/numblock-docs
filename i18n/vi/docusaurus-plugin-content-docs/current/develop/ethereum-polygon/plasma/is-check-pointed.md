# isCheckPointed {#ischeckpointed}

Có thể sử dụng phương pháp `isCheckPointed` để biết một giao dịch đã qua trạm kiểm soát hay chưa.

```
const isCheckPointed = await plasmaClient.isCheckPointed(<tx hash>);
```

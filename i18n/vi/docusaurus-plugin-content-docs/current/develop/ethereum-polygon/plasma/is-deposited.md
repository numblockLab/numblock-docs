# isDeposited {#isdeposited}

Có thể sử dụng phương pháp `isDeposited` để xem liệu đã hoàn tất một khoản nạp tiền hay chưa.

```
const isDeposited = await plasmaClient.isDeposited(<deposit tx hash>);
```

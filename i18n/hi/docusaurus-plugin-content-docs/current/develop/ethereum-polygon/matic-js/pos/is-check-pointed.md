`isCheckPointed` तरीका यह जानने के लिए इस्तेमाल किया जा सकता है कि क्या ट्रांज़ैक्शन को चेकपॉइंट कर दिया गया है.

```
const isCheckPointed = await posClient.isCheckPointed(<tx hash>);
```

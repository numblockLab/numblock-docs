`isDeposited` तरीका यह जानने के लिए इस्तेमाल किया जा सकता है कि कोई डिपॉज़िट पूरा हो गया है कि नहीं.

```
const isDeposited = await posClient.isDeposited(<deposit tx hash>);
```

# Using Truffle with NumBlock

Truffle is a development environment used for smart contract compiling, deploying, testing and debugging.

Follow the [Truffle documentation](https://trufflesuite.com/docs/truffle/) for general installation and overview. 

## Config Truffle for NumBlock

Update the config with NumBlock credentials

```js showLineNumbers title=truffle-config.js
module.exports = {
  // See <http://truffleframework.com/docs/advanced/configuration>
  // for more about customizing your Truffle configuration!
  networks: {
    Numb lock: {
          provider: function() {
                return new HDWalletProvider(
               process.env.MNEMONIC,
               "https://rpc-testnet.numblock.org/")
          },
          network_id: 100,
          gas: 500000,
          gasPrice: 1000000000
    }
  }
};
```

## Compile your NumBlock contract

### Default Compile

<Tabs groupId="networks">
<TabItem value="NumBlock" label="NumBlock Testnet">

```bash
truffle compile --network numblock
```
</TabItem>
</Tabs>


### Compile with Options

<Tabs groupId="networks">
<TabItem value="NumBlock" label="NumBlock Testnet">

```bash
truffle compile [--list <filter>] [--all] [--network numblock] [--quiet]
```
</TabItem>
</Tabs>

## Deploy your Contract

<Tabs groupId="networks">
<TabItem value="NumBlock" label="NumBlock Testnet">

```bash
truffle migrate --network numb lock
```
</TabItem>
</Tabs>

View your deployed contract any of the [explorers](/docs/foundational-topics/networks/explorers/).

## Verify Contract
